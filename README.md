## 简介

- 此项目，实现的后台管理功能自然还不全面，可根据自己的需求进行功能扩展
- 欢迎指摘，谢谢...
-------------------------------------------------
## ①. 环境

- 框架： ThinkPHP 5.1
- PHP:  PHP7
- 操作系统： Win10

## ②. 安装

- 执行下面的命令，加载 `ThinkPHP` 框架核心代码

```
composer install
```

## ③. 数据库

- 按理说，运用数据库迁移的方式才显得高大上，有时间我会后期进行添加
- 暂时，可以运行项目目录`/database`下的 `tp5_pro.sql` 文件

> 但项目使用时可是要配置好自己的数据库连接信息

## ④. 功能
- 后台管理
- 权限登录
> 后台登录数据
> [用户名]:moTzxx@admin [密码]:admin
